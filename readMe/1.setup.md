# Springboot setup
- Install Java Development Kit (JDK) 11 or higher.
- Install an IDE (IntelliJ IDEA, Eclipse, or VS Code recommended).
- Install Maven or Gradle build tool.
  - brew install maven
  - brew install gradle
- Install Git for version control.
- (Optional) Install Docker for containerization.


## Create the project
- You can generate a project using start.spring.io, or create manually. Minimal coordinates:
- Group: com.example
- Artifact: bookapi
- Packaging: jar
- Java: 17
- Dependencies: Spring Web, Spring Data JPA, H2 Database, Validation (spring-boot-starter-validation)

## Install the extensions in VS code
- Install these extensions
- Search in Extensions sidebar (Ctrl+Shift+X):
✅ Spring Boot Extension Pack
✅ Spring Initializr Java Support
✅ Debugger for Java
✅ Language Support for Java by Red Hat
✅ Java Test Runner
✅ Maven for Java
✅ Gradle for Java-

```
java -version

mvn -v

```

## How to start and stop the Spring Boot application
- To start the application, run:
- Using Maven:
```
./mvnw spring-boot:run
```
## To stop the Server
- lsof -i :8080
- kill -9 <PID>]
- Using Gradle:
```
./gradlew bootRun
```

# To Start Or stop Spring boot application

2) Buildship Gradle Integration
